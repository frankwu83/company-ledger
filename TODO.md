## 部署建议

### 推荐云服务方案

| 服务 | 推荐配置 | 预估费用 |
|------|---------|---------|
| **云服务器** | 2核4G，CentOS/Ubuntu | ￥100-200/月 |
| **数据库** | 云数据库 RDS（MySQL/PostgreSQL） | ￥50-100/月 |
| **域名** | .com/.cn 域名 | ￥50-100/年 |
| **SSL证书** | 免费 Let's Encrypt 或付费 | 免费-￥500/年 |
| **对象存储** | 备份文件存储 | 按量计费 |

### 部署架构

```
用户 → 域名 → CDN（可选）→ Nginx → 前端静态资源
                                    ↓
                              Nginx 反向代理 → Node.js 后端 → 数据库
```

### 技术选型建议

| 组件 | 推荐方案 |
|------|---------|
| **服务器** | Ubuntu 22.04 LTS |
| **Node.js 版本** | v18 LTS |
| **进程管理** | PM2 |
| **Web 服务器** | Nginx |
| **数据库** | PostgreSQL 14+ |
| **SSL** | Let's Encrypt (Certbot) |
