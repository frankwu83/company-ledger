# Company Ledger（公司台账）

公司账目记录系统，用于台账的收集和整理，生成月度、季度和年度收支情况报告。

## 技术栈

- **前端**: Vue 3 + Vite + Element Plus
- **后端**: Node.js + Express
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **ORM**: Prisma
- **图表**: Chart.js

## 功能特性

- 📊 收入/支出台账录入
- 📁 分类管理
- 💰 账户管理
- 📈 月度/季度/年度报告
- 📤 Excel 导出
- 🔍 数据查询与筛选

## 项目结构

```
company-ledger/
├── frontend/          # Vue 3 前端
├── backend/           # Node.js 后端
├── database/          # 数据库模型
└── README.md
```

## 快速开始

### 安装依赖

```bash
# 后端
cd backend
npm install

# 前端
cd frontend
npm install
```

### 启动开发服务器

```bash
# 后端 (端口 3000)
cd backend
npm run dev

# 前端 (端口 5173)
cd frontend
npm run dev
```
